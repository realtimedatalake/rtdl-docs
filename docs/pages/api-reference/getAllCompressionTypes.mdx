# getAllCompressionTypes
**GET**  
**Service & Port:** config:80 -- e.g. http://localhost:80/getAllCompressionTypes  
`getAllCompressionTypes` gets all `compression_type` records.
  * Example `getAllCompressionTypes` curl call.  
    ```
    curl --location --request GET 'http://localhost:80/getAllCompressionTypes'
    ```
