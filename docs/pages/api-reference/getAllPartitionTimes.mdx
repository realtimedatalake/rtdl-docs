# getAllPartitionTimes
**GET**  
**Service & Port:** config:80 -- e.g. http://localhost:80/getAllPartitionTimes  
`getAllPartitionTimes` gets all `partition_time` records.
  * Example `getAllPartitionTimes` curl call.  
    ```
    curl --location --request GET 'http://localhost:80/getAllPartitionTimes'
    ```
