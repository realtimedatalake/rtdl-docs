# deleteStream
**DEL**  
**Service & Port:** config:80 -- e.g. http://localhost:80/deleteStream  
**Required fields:** `stream_id`  
`deleteStream` deletes a `stream` configuration record.
  * Example `deleteStream` call body.  
    ```
    {
      "stream_id": "[stream_id]"
    }
    ```
  * Example `deleteStream` curl call.  
    ```
    curl --location --request DELETE 'http://localhost:80/deleteStream' \
    --header 'Content-Type: application/json' \
    --data-raw '{
        "stream_id": "[stream_id]"
    }'
    ```
