# createStream
**POST**  
**Required fields:** `message_type`, `folder_name`  
`createStream` creates a `stream` configuration record that defines a data source 
a storage destination as well as parameters for the data lake that will be created.
  * Example `createStream` call for creating a local data lake.  
    ```
    {
      "active": true,
      "message_type": "test-msg-local",
      "file_store_type_id": 1,
      "folder_name": "testFolderLocal",
      "partition_time_id": 1,
      "compression_type_id": 1
    }
    ```
  * Example `createStream` call for creating a data lake on AWS S3.  
    ```
    {
      "active": true,
      "message_type": "test-msg-aws",
      "file_store_type_id": 2,
      "region": "us-west-1",
      "bucket_name": "testBucketAWS",
      "folder_name": "testFolderAWS",
      "partition_time_id": 1,
      "compression_type_id": 1,
      "aws_access_key_id": "[aws_access_key_id]",
      "aws_secret_access_key": "[aws_secret_access_key]"
    }
    ```
