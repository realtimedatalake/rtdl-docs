# getAllFileStoreTypes
**GET**  
**Service & Port:** config:80 -- e.g. http://localhost:80/getAllFileStoreTypes  
`getAllFileStoreTypes` gets all `file_store_type` records.
  * Example `getAllFileStoreTypes` curl call.  
    ```
    curl --location --request GET 'http://localhost:80/getAllFileStoreTypes'
    ```
