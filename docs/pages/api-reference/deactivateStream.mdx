# deactivateStream
**PUT**  
**Service & Port:** config:80 -- e.g. http://localhost:80/deactivateStream  
**Required fields:** `stream_id`  
`deactivateStream` sets the `active` value in a `stream` configuration record to false.
  * Example `deactivateStream` call body.  
    ```
    {
      "stream_id": "[stream_id]"
    }
    ```
  * Example `deactivateStream` curl call .  
    ```
    curl --location --request PUT 'http://localhost:80/deactivateStream' \
    --header 'Content-Type: application/json' \
    --data-raw '{
        "stream_id": "[stream_id]"
    }'
    ```
