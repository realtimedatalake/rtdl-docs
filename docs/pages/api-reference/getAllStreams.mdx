# getAllStreams
**GET**  
**Service & Port:** config:80 -- e.g. http://localhost:80/getAllStreams  
`getAllStreams` gets all `stream` configuration records.
  * Example `getAllStreams` curl call.  
    ```
    curl --location --request GET 'http://localhost:80/getAllStreams'
    ```
